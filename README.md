# this is me - Astro Static Site

Powers [thisisme.nz](https://www.thisisme.nz/)

## Getting Started

1. Install dependencies:

```bash
pnpm install
```

2. Run the development server:

```bash
pnpm dev
```

3. Build for production:

```bash
pnpm run build
```

4. Preview production build:

```bash
pnpm run preview
```
